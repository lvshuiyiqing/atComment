public class RandoopAllClasses
extends Object

Runs Randoop on a all the classes (together) of a given subject program. This command must be run from $RANDOOP_HOME/systemtests. The command is run as follows: java randoop.experiments.RandoopOneClass The command assumes: + The file experiments/.experiment exists and specifies a subject program property file. The command reads in the property file and uses it to create the appropriate classpath and other parameters.
