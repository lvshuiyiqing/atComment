Interface ExecutionVisitor

All Known Implementing Classes:
ContractCheckingVisitor, CovWitnessHelperVisitor, DummyVisitor, MultiVisitor, RegressionCaptureVisitor, ReplayVisitor
public interface ExecutionVisitor

A visitor that is invoked as the execution of a sequence unfolds. Typically such a visitor adds decorations to the sequence based on checks of the runtime behavior. IMPORTANT: Implementing classes should have a zero-argument constructor.
